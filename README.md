# Projeto FoodChegou - SPA (Single Page Application) de aplicativo baseado no Ifood feito no framework Angular

Este projeto se basea em uma SPA (Single Page Application) de um aplicativo baseado no ifood Angular, feito para além de aprendizado, também
para dominar as tecnologias que estão presentes dentro do framework

---

### Padrão de Projeto

O projeto seguirá o padrão adotado pela documentação do Angular versão 22, trazendo novidades e formas eficientes de como construir aplicações web

---

### Comandos da aplicação

Inicialização da aplicação
```bash
# Iniciar a aplicação
npm start
```

```bash
# Iniciar a aplicação em modo de monitoramento 
npm watch
```
Testes da aplicação
```bash
# irá rodar os testes da aplicação 
npm test
```

```bash
# irá rodar os testes da aplicação  
npm run e2e
```
Colocando o aplicativo para produção
```bash
# irá rodar o build da aplicação  
npm build
```
